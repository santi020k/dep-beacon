# Dep Beacon for VS Code

Dep Beacon adds dependency version and security signals to npm manifests.

## Features

- CodeLens actions for next patch, next minor, next major, and latest dependency updates.
- Colored inline status for up-to-date, outdated, vulnerable, and invalid dependency ranges.
- Diagnostics for missing packages, unpublished version floors, outdated ranges, and OSV vulnerabilities.
- pnpm workspace catalog and named catalog support.
- `package.json` sorting for dependency-like sections.
- Cache clearing, prerelease toggles, package page links, and install commands.

Open `package.json` or `pnpm-workspace.yaml` and the extension starts automatically.
